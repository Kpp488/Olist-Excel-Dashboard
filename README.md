# 🛒 Olist E-Commerce Sales Dashboard (Excel Project)

Hi!  Thanks for stopping by. I'm Kamlesh, and this is my data analytics project using the real-world **Olist** dataset (a major e-commerce company in Brazil).

I built this interactive Excel dashboard to show how I can take raw, messy data spread across multiple files and turn it into clear business decisions. 

# 🛠️ What I Used
* **Tool:** Microsoft Excel (Advanced)
* **Techniques:** Power Query (to clean and connect 9 different CSV files), Pivot Tables, VLOOKUP/XLOOKUP, Data Modeling, and Interactive Slicers.

#  About the Data
The original dataset was huge and scattered across 9 separate raw CSV files (like customers, orders, payments, sellers, and products). Instead of getting confused, I connected all these tables logically to find out exactly what's happening in the business.

#  Key Things I Found Out (Business Insights)
1. **Top Revenue Generators:** Identified which specific product categories are actually bringing in the most money.
2. **Delivery Delays:** Analyzed how shipping times change depending on the state, and where the company is struggling with late deliveries.
3. **Customer Payment Habits:** Mapped out how most customers prefer to pay (Credit Card vs. Vouchers) to help target future offers.
